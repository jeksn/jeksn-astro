# {frontmatter.title}
### {frontmatter.date}

This program generates a 
random password of length 10 by selecting random characters from a set of allowed characters. The `randomChar` function selects a random character from the set and returns it as a `rune`. The `main` function uses `randomChar` to generate 10 random characters and concatenates them into a string, which it then prints to the console.

&nbsp;
&nbsp;

```go
package main

import (
    "fmt"
    "math/rand"
    "time"
)

const passwordLength = 10

func main() {
    rand.Seed(time.Now().UnixNano())

    var password string
    for i := 0; i < passwordLength; i++ {
        password += string(randomChar())
    }

    fmt.Println(password)
}

func randomChar() rune {
    chars := "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ0123456789"
    return rune(chars[rand.Intn(len(chars))])
}
```
